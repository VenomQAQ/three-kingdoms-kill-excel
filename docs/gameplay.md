# 玩法设计 — 身份局（界限突破标准版）

## 1. 游戏概述

- **模式**：身份局（标准模式）
- **人数**：2 ~ 10 人（推荐 8 人）
- **版本**：《三国杀标准版·界限突破》（2014）
- **胜利条件**：见身份牌说明

## 3. 游戏准备

1. 确定座位顺序（顺时针）
2. 分发身份牌（主公亮明，其余暗置）
3. 主公先选将或随机分发武将牌（本项目建议：**随机 3 将选 1** 或 **Room 房主配置**）
4. 分发起始手牌：每人 **4 张**
5. 将游戏牌洗匀置牌堆

**牌堆构成（109 张）**：

- 基本牌 53 张
- 锦囊牌 36 张
- 装备牌 20 张（含木牛流马 1 张）

---

## 4. 回合流程

每个回合按以下阶段顺序进行（若某阶段被跳过则进入下一阶段）：

```
开始 → 判定 → 摸牌 → 出牌 → 弃牌 → 结束
```

### 4.1 开始阶段

- 部分技能在此触发（如【洛神】不在此，在摸牌阶段前/替代摸牌等以技能描述为准）

### 4.2 判定阶段

- 处理判定区内的延时锦囊：**【乐不思蜀】**、**【兵粮寸断】**、**【闪电】**
- 判定顺序：按放置先后顺序，先放先判
- 判定：从牌堆顶摸一张判定牌，花色/点数生效后通常进入弃牌堆

### 4.3 摸牌阶段

- 默认摸 **2 张**牌
- 可被技能替换或额外摸牌（如界周瑜·英姿）
- 【兵粮寸断】跳过摸牌阶段（在判定阶段处理）

### 4.4 出牌阶段

- 可任意使用手牌或技能（除非有限制）
- 默认 **无出牌次数限制**，但【杀】默认每回合限 **1 次**（武器/技能可改变）
- 可使用装备牌置入装备区

### 4.5 弃牌阶段

- 手牌数不得超过 **当前体力值**
- 超出部分须自行弃置

### 4.6 结束阶段

- 部分技能在此触发

## 5. 基本行动规则

### 5.1 使用【杀】

1. 选择攻击范围内的一名其他角色为目标
2. 目标可打出【闪】抵消
3. 若未抵消，目标受到 1 点伤害
4. 默认每回合对同一目标只能使用一张【杀】（除非诸葛连弩等）

### 5.2 使用【闪】

- 只能当【杀】或万箭齐发等指定时打出
- 不能主动使用

### 5.3 使用【桃】

- 出牌阶段对自己使用：回复 1 点体力
- 濒死求桃：任意角色可打出/使用【桃】救活（体力为 0 时）

### 5.4 濒死与死亡

1. 体力降至 0 或更低时进入濒死
2. 从当前回合角色开始逆时针询问是否使用【桃】
3. 若体力仍 ≤ 0，则死亡
4. 死亡：身份公开（若未公开）、武将置灰、手牌与装备全部弃置
5. **奖惩**：
  - 杀死 **反贼** 的角色摸 **3 张**牌
  - 杀死 **忠臣** 且凶手是 **主公**，主公须弃置 **所有**手牌和装备

### 5.5 距离与范围

- **距离**：两角色间最短顺时针/逆时针座位差（取较小值）
- **攻击范围**：默认 1；武器可 +N；部分技能可改变
- **距离计算修正**：+1 马、-1 马、马术、义从等

### 5.6 判定区与延时锦囊

- 延时锦囊置入目标 **判定区**（非手牌）
- 【乐不思蜀】：若判定非红桃，跳过出牌阶段
- 【兵粮寸断】：若判定非梅花，跳过摸牌阶段
- 【闪电】：判定黑桃 2~9 则受 3 雷伤，否则传给下家

### 5.7 【无懈可击】

- 可响应普通锦囊、延时锦囊与群体锦囊
- 本项目当前 sandbox 中按座次逐个询问拥有【无懈可击】的角色
- 当一张锦囊存在多个目标时，可选择抵消单个目标，或在允许时直接抵消全部目标效果

## 7. 胜负条件


| 身份  | 胜利条件                                                                                |
| --- | ----------------------------------------------------------------------------------- |
| 主公  | 消灭所有反贼和内奸                                                                           |
| 忠臣  | 主公获胜时，忠臣一同获胜                                                                        |
| 反贼  | 主公死亡                                                                                |
| 内奸  | 消灭除自己外所有角色，且最后与主公单挑时杀死主公（或成为最后存活者，依 house rule；**默认采用官方**：内奸需个人独胜，主公死时若仅剩内奸与反贼则反贼胜） |


**官方内奸细则**：

- 主公死亡时，若还有反贼在场，反贼胜（内奸失败）
- 内奸必须成为最后唯一存活者才算胜利（部分场次简化：主公死后只剩内奸则内奸胜 — 房间可配置）

## 8. 本项目在线规则扩展

### 8.1 房间流程

```
创建房间 → 分享 8 位房间号 → 玩家加入 → 准备 → 房主开始
→ 选将/分将 → 发身份 → 发初始手牌 → 从主公开始游戏
```

### 8.2 超时处理


| 场景        | 默认时间 | 超时行为        |
| --------- | ---- | ----------- |
| 出牌阶段      | 30s  | 自动结束出牌      |
| 响应（出闪/无懈） | 15s  | 视为不响应       |
| 选目标/选牌    | 20s  | 随机合法目标 / 取消 |
| 选将        | 60s  | 随机分配        |


### 8.3 断线

- 断线后保留座位 **120s**，可重连回同一 seat
- 超时未重连：进入 **托管**（仅自动出闪/桃保命，不主动进攻）
- **模拟测试房**：按昵称匹配已断线的真人玩家，复用原角色 id；对局进行中会迁移引擎内 player id 与 prompt 上下文，避免刷新后重复创建角色

### 8.5 模拟测试房（70755712）

单连接可操控多名角色（含虚拟角色），用于规则验证：

| 能力 | 说明 |
|------|------|
| 回合流程 | 判定 → 摸牌 → 出牌 → 弃牌 → 结束，自动切换下一角色操控 |
| 出牌弹窗 | 确认出牌 → 选目标 → 响应（杀/闪）→ 伤害结算 |
| 锦囊响应 | 普通锦囊/延时锦囊/AOE 可进入【无懈可击】询问链 |
| 受伤后技能 | 仅**受伤角色**可触发（如界司马懿【反馈】、界曹操【奸雄】） |
| 选区域牌 | 【过河拆桥】弃置目标区域一张牌；【顺手牵羊】获得目标区域一张牌 |
| 手牌隐私 | 选对手手牌时仅显示「手牌 1」「手牌 2」…，顺序随机；装备区明牌 |
| 弃牌阶段 | 手牌超过体力上限时弹窗选牌弃置 |
| 改判 | 判定阶段【鬼才】可打出一张手牌代替判定结果 |
| 主动技能 | 当前已接线可在出牌阶段主动操作【仁德】、【制衡】、【荐言】 |

### 8.4 聊天规范

- 禁止发送完整手牌信息（系统过滤 `[card:` 等）
- 允许战术交流；身份局允许「语聊」但不可明牌身份（系统不做限制，靠玩家自觉）

## 9. 与 2013 标准版的主要差异（界限突破）

游玩本项目时，以下武将/牌面采用 **界** 技能，详见 [characters.md](./cards/characters.md)：

- 18 名武将技能已调整（如界曹操奸雄、界司马懿反馈等）
- 新增：界李典、界徐庶、界公孙瓒
- 未改将：孙权、甄姬、诸葛亮、孙尚香（仍用原标准技能）
- 新增牌：**木牛流马**（宝物）
