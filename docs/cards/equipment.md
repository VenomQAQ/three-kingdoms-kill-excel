## 4. 宝物（界限突破新增）

### 【木牛流马】×1

| 属性 | 值 |
|------|-----|
| 槽位 | 宝物区 |
| 效果 | 出牌阶段限一次：将一张手牌扣置于木牛下称为「粮」；可将此装备移动至其他角色装备区；可使用或打出「粮」 |
| 牌面 | 界限突破扩展单独印刷 |

**规则要点**：

- 「粮」对你可见
- 装备区木牛移动时，粮始终扣于其下
- 木牛离开装备区进入非交换场景时，粮与木牛一同置入弃牌堆

## 5. 配置示例

```yaml
# config/cards/equipment/qinggang_jian.yml
id: qinggang_jian
name: 青釭剑
type: equipment
subType: weapon
attackRange: 2
passive:
  - timing: ON_SHA_HIT
    action: ignoreArmor
```

```yaml
# config/cards/equipment/muniu_liuma.yml
id: muniu_liuma
name: 木牛流马
type: equipment
subType: treasure
active:
  - timing: PHASE_PLAY
    limit: 1
    action: storeGrainAsLiang
  - action: moveEquipmentToOther
effects:
  - canUseLiangAsHand: true
```

---

## 6. 装备区规则摘要

- 各子类型装备区 **至多 1 件**（武器/防具/+1马/-1马/宝物）
- 新装备同类型时，原装备弃置
- 装备牌可在出牌阶段置入（视为使用，不占用「杀」次数）
