# 武将牌收录 — 界限突破（30 将）

> 版本：《三国杀标准版·界限突破》（2014）  
> 编号格式：`势力代码 + 序号`（如 WEI 001）

## 图例

| 标记 | 含义 |
|------|------|
| 【界】 | 界限突破修改技能 |
| 原 | 沿用 2013 标准版技能，未做界改 |
| 主公技 | 仅主公可用 |
| 锁定技 | 满足条件须发动/持续生效 |
| 限定技 | 整局限一次 |
| 觉醒技 | 满足条件后永久获得新技能 |

## 魏国（8 将）

### 界曹操 · 魏武帝 `WEI 001`

| 属性 | 值 |
|------|-----|
| 势力 | 魏 |
| 体力 | 4 |
| 技能 | 奸雄、护驾【主公技】 |

**奸雄** — 当你受到伤害后，你可以获得造成此伤害的牌并摸一张牌。

**护驾**【主公技】— 当你需要使用或打出【闪】时，你可以令其他魏势力角色选择是否打出一张【闪】（视为由你使用或打出）。

---

### 界司马懿 · 狼顾之鬼 `WEI 002`

| 属性 | 值 |
|------|-----|
| 势力 | 魏 |
| 体力 | 3 |
| 技能 | 反馈、鬼才 |

**反馈** — 当你受到 1 点伤害后，你可以获得伤害来源的一张牌。

**鬼才** — 当一名角色的判定牌生效前，你可以打出一张牌代替之。

---

### 界夏侯惇 · 独眼的罗刹 `WEI 003`

| 属性 | 值 |
|------|-----|
| 势力 | 魏 |
| 体力 | 4 |
| 技能 | 刚烈、清俭 |

**刚烈** — 当你受到 1 点伤害后，你可以进行判定：若结果为红色，你对伤害来源造成 1 点伤害；若结果为黑色，你弃置其一张牌。

**清俭** — 当你于摸牌阶段外获得牌后，你可以将其中任意张牌交给其他角色。

---

### 界张辽 · 晋阳侯 `WEI 004`

| 属性 | 值 |
|------|-----|
| 势力 | 魏 |
| 体力 | 4 |
| 技能 | 突袭 |

**突袭** — 摸牌阶段，你可以少摸任意张牌并获得等量其他角色各一张手牌。

---

### 界许褚 · 牟乡侯 `WEI 005`

| 属性 | 值 |
|------|-----|
| 势力 | 魏 |
| 体力 | 4 |
| 技能 | 裸衣 |

**裸衣** — 摸牌阶段，你可以跳过摸牌并亮出牌堆顶三张牌，然后获得其中的基本牌、武器牌和【决斗】，且直到下回合开始，你使用【杀】或【决斗】造成的伤害 +1。

### 界郭嘉 · 英才天妒 `WEI 006`

|------|-----|
| 势力 | 魏 |

| 技能 | 天妒、遗计 |

**天妒** — 当你的判定牌生效后，你可以获得此牌。

**遗计** — 当你受到 1 点伤害后，你可以摸两张牌，然后将至多两张手牌交给一至两名其他角色。

**洛神** — 准备阶段，你可以进行判定，若结果为黑色，你获得此牌并可以重复此流程。

**忘隙** — 当你对其他角色造成 1 点伤害后，或受到其他角色造成的 1 点伤害后，你可以与该角色各摸一张牌。

## 蜀国（8 将）

### 界刘备 · 汉昭烈帝 `SHU 001`

| 属性 | 值 |
|------|-----|
| 势力 | 蜀 |
| 体力 | 4 |
| 技能 | 仁德、激将【主公技】 |

**仁德** — 出牌阶段每名角色限一次，你可以将任意张手牌交给一名其他角色；当你于此阶段内第二次以此法给牌后，你可以视为使用一张基本牌。

**激将**【主公技】— 当你需要使用或打出【杀】时，你可以令其他蜀势力角色选择是否打出一张【杀】（视为由你使用或打出）。

---

### 界关羽 · 壮缪侯 `SHU 002`

| 属性 | 值 |
|------|-----|
| 势力 | 蜀 |
| 体力 | 4 |
| 技能 | 武圣、义绝 |

**武圣** — 你可以将一张红色牌当【杀】使用或打出。

**义绝** — 出牌阶段限一次，你可以与一名角色拼点：若你赢，其本回合不能使用或打出手牌且非锁定技失效；若你没赢，你可以令其回复 1 点体力。

---

### 界张飞 · 新亭侯 `SHU 003`

| 属性 | 值 |
|------|-----|
| 势力 | 蜀 |
| 体力 | 4 |
| 技能 | 咆哮、替身 |

**咆哮**【锁定技】— 你使用【杀】无次数限制。

**替身**【限定技】— 准备阶段，你可以将体力回复至等同于上回合结束时的体力，然后摸 X 张牌（X 为回复的体力值）。

---

### 诸葛亮 · 忠武侯 `SHU 004` 【原】

| 属性 | 值 |
|------|-----|
| 势力 | 蜀 |
| 体力 | 3 |
| 技能 | 观星、空城 |

**观星** — 准备阶段，你可以观看牌堆顶五张牌（存活角色 ≤3 时为三张），以任意顺序置于牌堆顶或牌堆底。

**空城**【锁定技】— 若你没有手牌，你不能成为【杀】或【决斗】的目标。

---

### 界赵云 · 虎威将军 `SHU 005`

| 属性 | 值 |
|------|-----|
| 势力 | 蜀 |
| 体力 | 4 |
| 技能 | 龙胆、涯角 |

**龙胆** — 你可以将【杀】当【闪】、【闪】当【杀】使用或打出。

**涯角** — 当你于回合外使用或打出手牌时，你可以展示牌堆顶一张牌：若与此牌类别相同，可将展示牌交给一名角色；若不同，可将展示牌置入弃牌堆。

---

### 界马超 · 一骑当千 `SHU 006`

| 属性 | 值 |
|------|-----|
| 势力 | 蜀 |
| 体力 | 4 |
| 技能 | 马术、铁骑 |

**马术**【锁定技】— 你计算与其他角色的距离 -1。

**铁骑** — 当你使用【杀】指定目标后，可令其本回合非锁定技失效，然后你判定：除非其弃置一张与结果花色相同的牌，否则不能使用【闪】响应此【杀】。

### 界黄月英 · 归隐的杰女 `SHU 007`

| 属性 | 值 |
|------|-----|
| 势力 | 蜀 |
| 体力 | 3 |
| 技能 | 集智、奇才 |

**集智** — 当你使用普通锦囊牌时，你可以摸一张牌；若摸到的为基本牌，可弃置之并本回合手牌上限 +1。

**奇才**【锁定技】— 你使用锦囊牌无距离限制；其他角色不能弃置你装备区里的防具和宝物牌。

---

### 界徐庶 · 化剑为犁 `SHU 017` 【新】

| 属性 | 值 |
|------|-----|
| 势力 | 蜀 |
| 体力 | 4 |
| 技能 | 诛害、潜心、荐言 |

**诛害** — 其他角色的结束阶段，若其本回合造成过伤害，你可以对其使用一张【杀】。

**潜心**【觉醒技】— 当你造成伤害后，若你已受伤，你减 1 点体力上限并获得「荐言」。

**荐言** — 出牌阶段限一次，你可以声明一种牌的类别或颜色，亮出牌堆顶符合声明的第一张牌并交给一名男性角色。

## 实现优先级建议

| 优先级 | 武将 | 原因 |
|--------|------|------|
| P0 | 曹操、赵云、吕布、华佗 | 高频、技能代表性强 |
| P1 | 周瑜、张辽、马超、甘宁 | 含复杂交互 |
| P2 | 徐庶、李典、公孙瓒 | 觉醒/特殊流程 |
| P3 | 原技能将（甄姬、诸葛亮等） | 可复用标准 Handler |

> 技能描述以 2014 界限突破首发版为准；后续实现时建议对照《三国杀官方规则集》做最终校验。
