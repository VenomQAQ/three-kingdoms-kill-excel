# 锦囊牌收录

> 共 **36 张**（标准 108 张中的锦囊部分）

## 分类

| 子类 | 说明 | 代表 |
|------|------|------|
| 即时锦囊 | 使用即结算 | 过河拆桥、决斗 |
| 延时锦囊 | 置入判定区 | 乐不思蜀、闪电 |
| 响应锦囊 | 可抵消其他锦囊 | 无懈可击 |

---

## 1. 即时锦囊

### 【决斗】×3

| 属性 | 值 |
|------|-----|
| 目标 | 一名其他角色 |
| 效果 | 与目标轮流打出【杀】，先不出者受 1 伤害 |
| 牌面 | ♠A, ♣A, ♦A |

### 【过河拆桥】×6

| 属性 | 值 |
|------|-----|
| 目标 | 距离 1 以内的一名其他角色 |
| 效果 | 弃置目标区域里的一张牌（手牌/装备/判定） |
| 牌面 | ♠3, ♠4, ♣3, ♣4, ♠Q, ♥Q |
| **引擎实现** | `select_zone_card`：手牌匿名+乱序，装备明牌，玩家自选 |

### 【顺手牵羊】×5

| 属性 | 值 |
|------|-----|
| 目标 | 距离 1 以内的一名其他角色 |
| 效果 | 获得目标区域里的一张牌 |
| 牌面 | ♠3, ♠4, ♠J, ♦3, ♦4 |
| **引擎实现** | 同过河拆桥，效果为获得所选牌入手牌 |

### 【无中生有】×4

| 属性 | 值 |
|------|-----|
| 目标 | 自己 |
| 效果 | 摸 2 张牌 |
| 牌面 | ♥7, ♥8, ♥9, ♥J |

### 【五谷丰登】×2

| 属性 | 值 |
|------|-----|
| 目标 | 所有角色 |
| 效果 | 亮出牌堆顶等于角色数的牌，每人选一张获得 |
| 牌面 | ♥3, ♥4 |

### 【桃园结义】×1

| 属性 | 值 |
|------|-----|
| 目标 | 所有角色 |
| 效果 | 所有角色回复 1 点体力 |
| 牌面 | ♥A |

### 【借刀杀人】×2

|------|-----|

## 5. 引擎实现要点

| 牌名 | Handler 层级 | 关键逻辑 |
|------|-------------|----------|
| 决斗 | L2 | 轮流响应循环，直到一方不出杀 |
| 南蛮/万箭 | L2 | 按座位顺序依次响应 |
| 五谷丰登 | L2 | 亮牌 → 主公先选 → 逆时针选牌 |
| 借刀杀人 | L3 | 两目标选择 + 强制杀或收武器 |
| 无懈可击 | L2 | 插入响应链，支持对无懈无懈 |
| 乐/兵粮/闪电 | L2 | 判定区 + 判定阶段触发 |
| 过河/顺手 | L2 | 选区域 + 选牌（手牌需随机/指定策略） |

### 配置示例

```yaml
# config/cards/trick/nanman_ruqin.yml
id: nanman_ruqin
name: 南蛮入侵
type: trick
subType: aoe
targeting:
  selector: allOthers
  filter: { alive: true }
effects:
  - action: sequentialResponse
    params:
      responseCard: sha
      damageOnFail: 1
      order: turnOrder
```
