## 2. 正式房间 Gateway

| ID | 用例 | 步骤 | 预期 |
|----|------|------|------|
| GW-01 | room:start 开局 | 全员准备后房主开始 | game:started + 每人收到过滤后 room:state |
| GW-02 | game:playCard | 轮到自己出牌阶段打【杀】 | 状态同步，他人看不到手牌 |
| GW-03 | game:submitResponse | 被【杀】时出【闪】 | 仅被询问者收到 response prompt |
| GW-04 | game:sync 重连 | 断线重连 emit game:sync | 收到完整过滤状态 |
| GW-05 | game:finished | 一方获胜 | game:finished + victory 信息 |

## 3. AOE 锦囊

| AOE-01 | 南蛮入侵 | 对3人使用，1人不出【杀】 | 按座次询问，失败者受伤 |
