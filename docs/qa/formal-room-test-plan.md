## 0. 正式房等待大厅

| ID | 用例 | 步骤 | 预期 |
|----|------|------|------|
| LOBBY-01 | Ribbon 准备 | 登录后创建正式房，点击「准备」 | 本人「准备」列变为「已准备」 |
| LOBBY-02 | 公式栏 /ready | 在房间内输入 `/ready` | 准备状态切换 |
| LOBBY-03 | 表格单击准备 | 单击本人行的「准备」列（蓝色下划线） | 准备状态切换 |
| LOBBY-04 | 测试房禁用准备 | 进入 70755712 | Ribbon「准备」为禁用；提示添加角色后模拟开局 |
| LOBBY-05 | 切控污染修复 | 测试房切换虚拟角色后离开，再进正式房点准备 | 准备作用于本人，不报「不在房间内」 |
| LOBBY-06 | 房主开始 | ≥2 人全员准备，房主点「开始」 | `game:started`，身份分配，进入对局表格 |
| LOBBY-07 | 非房主开始 | 非房主点「开始」 | 按钮禁用或提示仅房主可开始 |
| LOBBY-08 | 重连后准备 | 正式房等待中刷新页面，再点 Ribbon「准备」或本人 E 列 | 准备成功，不报「不在房间内」 |

## 2. 正式房间 Gateway

| ID | 用例 | 步骤 | 预期 |
|----|------|------|------|
| GW-01 | room:start 开局 | 全员准备后房主开始 | game:started + 每人收到过滤后 room:state |
| GW-02 | game:playCard | 轮到自己出牌阶段打【杀】 | 状态同步，他人看不到手牌 |
| GW-03 | game:submitResponse | 被【杀】时出【闪】 | 仅被询问者收到 response prompt |
| GW-04 | game:sync 重连 | 断线重连 emit game:sync | 收到完整过滤状态 |
| GW-05 | game:finished | 一方获胜 | game:finished + victory 信息 |
