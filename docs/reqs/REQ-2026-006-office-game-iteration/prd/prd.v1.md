# WPS 化游戏迭代 (REQ-2026-006)

## 1. 背景与目标

当前应用已经具备 WPS 表格伪装、大厅房间、三国杀、连连看和玩家资料能力，但下一轮体验存在四类断点：玩家战绩没有按游戏拆分；连连看简单难度格子偏少且主题不够丰富；顶部工作栏仍暴露“已登录/连接”类系统感信息；底部 Sheet 不能本地改名；大厅还不能创建“大富翁”房间。

本期目标是在继续保持“像 WPS Excel”的第一视觉前提下，让玩家能在同一套表格外壳中切换三国杀、连连看和世界版大富翁，并让账号信息、设置、资料与 Sheet 操作更像办公软件内的自然功能。

成功指标：
- 个人资料的战绩区能以 tab 区分三国杀、连连看、大富翁，空态清晰。
- 连连看三档难度的格子数与主题数增加，简单难度不低于 8x8。
- 顶部信息栏不再出现“已登录 xxx / 连接：已连接”文案，改为账号等级、昵称、金币、在线状态。
- Sheet 标签右键可本地重命名，刷新后仍保留。
- 大厅可创建世界版大富翁房间，房间内排列与三国杀房间基本一致，游戏主界面为 Excel 风格表格棋盘。
- 大富翁游玩不扣金币。

## 2. 用户与场景

目标用户：
- 已登录玩家：需要查看分游戏战绩、创建或加入不同游戏房间。
- 新手玩家：需要更容易识别连连看难度、主题和大富翁基础状态。
- 办公伪装场景玩家：需要所有主要界面保持 WPS 表格质感，避免露出明显游戏化入口。

关键场景：
- 玩家点击工作栏中的昵称，打开个人详情，并在战绩 tab 中切换三国杀、连连看、大富翁。
- 玩家在连连看 Sheet 选择更大棋盘和更多主题后开局。
- 玩家点击功能区“设置”，在弹窗中修改昵称、修改密码或浏览器标签页标题。
- 玩家右键底部 Sheet 标签，选择“重命名”，输入本地名称。
- 玩家在房间列表选择“大富翁”类型创建房间，进入与三国杀房间排列一致的等待房，开始后看到 Excel 风格世界版大富翁棋盘。

## 4. 交互设计

### 4.1 工作栏与个人详情

工作栏左侧显示账号胶囊：`Lv.1 昵称 · 100金币`。后跟 `当前状态：在线/离线`。昵称为按钮，点击后打开个人详情弹窗。未登录时显示 `未登录` 与登录按钮。

个人详情弹窗中，基础信息保持昵称、等级、金币；战绩区改为 tab。tab 内容结构一致：总局数、胜利、失败、胜率、最近更新。连连看可将挑战成功视为胜利，失败/超时视为失败；大富翁本期可展示 0 值并预留结算接入。

### 4.2 设置弹窗

功能区“开始”页的第一组按钮从“粘贴”改为“设置”。点击打开设置弹窗，弹窗包含三块操作：修改昵称、修改密码、浏览器标签页标题。保存标题后立即更新当前 `document.title`，并保存在本地。

### 4.3 Sheet 标签右键

用户右键底部 Sheet 标签时，在标签上方出现轻量菜单。菜单至少包含“重命名”。点击后弹出输入框或原生输入，用户确认后更新该 Sheet 标签显示名称并写入本地。空名称、过长名称取消或截断提示，不影响原始 Sheet ID。

### 4.4 连连看

连连看 Sheet 保持工具条 + 表格网格布局。难度调整为：简单 8x8、普通 10x10、困难 12x12。主题列表增加颜文字与 emoji 表情主题，图案在单元格内显示，不改变网格稳定尺寸。

### 4.5 世界版大富翁

房间列表顶部工具区提供游戏类型选择，默认三国杀，可切换为大富翁后创建房间。房间列表新增“类型”列。大富翁等待房沿用三国杀房间布局；开始后切到 Excel 风格棋盘：外围为世界城市路线，中间区域展示玩家资产、当前位置、当前回合、日志和操作。首期玩法支持掷骰、移动、购买空地、收取过路费、回合轮转和胜负占位。

## 5. 规则与约束

- 三国杀既有规则、出牌阶段、结算队列、响应窗口不在本期改动范围内。
- 连连看仍按现有路径连线判定，不引入对战模式。
- 大富翁仅做世界版首期，不扩展其他版本、股票、卡片、拍卖等复杂规则。
- 大富翁首期房间与状态可轻量实现，但服务端仍需作为房间状态来源，避免客户端单方改棋盘。
- 游玩免费，不扣除金币；资料中的金币只展示。

## 6. 非功能要求

- 性能：连连看 12x12 和大富翁棋盘渲染不得产生明显卡顿；棋盘单元格使用稳定尺寸，避免频繁布局抖动。
- 兼容：本地 Sheet 名称和浏览器标题保存在 localStorage；数据损坏时回退默认名称。
- 可测试：新增交互需有前端组件测试覆盖；房间类型与大富翁基础状态需有服务端单元测试或可构造测试覆盖。
- 视觉：控件密度、边框、标题行、行列头、底部 Sheet 必须保留 WPS Excel 风格。
