- 引擎：实现界徐庶【诛害】在造成过伤害的其他角色结束阶段可选发动，对其使用一张【杀】，并在结算后回到当前角色结束流程。
- 引擎：确认界徐庶【潜心】已在造成伤害后按已受伤条件觉醒，减 1 点体力上限并获得【荐言】，且一局只触发一次。
