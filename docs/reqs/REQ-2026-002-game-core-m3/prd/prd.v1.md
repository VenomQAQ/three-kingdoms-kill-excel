# PRD · REQ-2026-002 · 游戏核心 M3 + 正式房大厅

- **版本**：v1
- **状态**：③开发中（功能流程以 [REQ-2026-003](../REQ-2026-003-ux-gameplay-polish/prd/prd.v1.md) 为准）
- **时间**：2026-07-03

## 1. 背景与目标

在 M1/M2（WPS 壳、测试房完整对局）与 REQ-2026-001（账号/大厅）完成后，优先补齐 **规则引擎 M3** 与 **正式房间等待大厅**，观战模式延后。

成功指标：

- 正式房：创建/加入后可正常 **准备 / 开始**，全员准备后房主可开局。
- M3 锦囊：【五谷丰登】【借刀杀人】可完整结算；【南蛮入侵】【万箭齐发】按 TargetQueue 顺序结算且先消耗手牌。
- M3 装备：【八卦阵】响应【杀】/【万箭】时可判定当闪；【木牛流马】可装备至宝物区。
- 测试房与正式房共用同一套 `SangokushiEngine` 规则。

## 2. 需求点

| 编号 | 需求点 | 优先级 |
|------|--------|--------|
| R-M3-01 | 【五谷丰登】亮出等于存活角色数的牌，按座次每人选一张 | P0 |
| R-M3-02 | 【借刀杀人】选持刀者+攻击范围内目标，持刀者出杀或失去武器 | P0 |
| R-M3-03 | AOE（南蛮/万箭）使用前先消耗锦囊，TargetQueue 逐目标响应 | P0 |
| R-M3-04 | 【八卦阵】需闪时可判定，红色视为打出【闪】 | P0 |
| R-M3-05 | 【木牛流马】宝物槽装备（粮机制后续 M4 扩展） | P1 |
| R-FR-01 | 正式房 Ribbon「准备」可用（非测试房） | P0 |
| R-FR-02 | 正式房大厅表格提示文案与测试房区分 | P1 |
| R-FR-03 | 正式房「准备」列单击切换准备状态（本人行，蓝色下划线样式） | P1 |
| R-FR-04 | `room:ready` / `room:start` 使用真实 `playerId`，不受 sandbox 切控污染 | P0 |
| R-FR-05 | 断线重连后客户端 `playerId` 与房间座位对齐（`auth:hello` + `room:state` 昵称回退） | P0 |

## 3. 非目标（本期不做）

- 观战模式
- 木牛流马「粮」扣置与转移完整交互
- 青龙偃月刀 / 贯石斧 / 方天画戟 / 麒麟弓 武器追伤流程
- M4 全 30 将技能
- E2E 自动化测试执行（开发完成后再测）

## 4. 验收标准

| AC | 描述 |
|----|------|
| AC-FR-01 | 2 人在正式房均可点击 Ribbon「准备」，状态同步为「已准备」 |
| AC-FR-02 | 从测试房切控虚拟角色后进入正式房，准备仍作用于本人座位 |
| AC-FR-03 | 刷新页面重连后，Ribbon「准备」与本人「准备」列单击均可切换状态 |
| AC-M3-01 | 使用【五谷丰登】后每名存活角色依次选一张亮牌入手 |
| AC-M3-02 | 【借刀杀人】持刀者不出杀时，使用者获得其武器 |
| AC-M3-03 | 【南蛮入侵】按座次询问，阵亡角色被跳过 |
| AC-M3-04 | 装备【八卦阵】的角色可对【万箭齐发】发动判定响应 |
