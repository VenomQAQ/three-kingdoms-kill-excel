# REQ-2026-008 · 大富翁界面优化与设置调整

## 背景

大富翁对局界面信息冗余、机会/命运卡展示干扰操作；设置项需与游戏场景解耦。

## 需求

| ID | 描述 | 优先级 |
|---|---|---|
| R1 | 「隐藏地块颜色」移至大富翁对局工具栏 | P0 |
| R2 | 机会/命运卡触发后不得卡住回合；卡面内容仅在日志展示 | P0 |
| R3 | 设置中老板键改为可配置快捷键，按下后切至「区域销售」Sheet | P0 |
| R4 | 大富翁对局时隐藏 Ribbon 技能区、InfoBar 操控信息、工具栏当前玩家/骰子/抽卡文案 | P1 |
| R5 | 地块上玩家 tag 移至右上角，超过 5 字截断 | P1 |
| R6 | 玩家资产表去除「资料」「状态」列；点击名称弹窗展示资产与地块详情；名称前在线/离线圆点 | P0 |
| R7 | 修改密码：关闭设置弹窗后独立弹窗，校验 QQ 邮箱 + 旧密码 | P0 |

## 验收

- AC1：大富翁工具栏可切换地块颜色显示
- AC2：抽机会/命运卡后回合正常推进，日志含卡面文案
- AC3：自定义快捷键可打开区域销售 Sheet
- AC4：对局时图 3 圈选 UI 不可见
- AC5：玩家 tag 在右上角且超长截断
- AC6：点击玩家名可查看持有地块等级与租金
- AC7：邮箱或旧密码不匹配时禁止改密
